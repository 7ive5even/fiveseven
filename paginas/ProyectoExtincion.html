Proyecto Extinción Z es un Juego de Rol con Dados creado por la Comunidad de FiveSeven.
Está pensado para jugar con mapas creados por uno mismo o el uso de Google Maps para moverse.

También los interiores pueden ser customizados para contener lugares donde obtener items mediante el sistema de Saqueo.
